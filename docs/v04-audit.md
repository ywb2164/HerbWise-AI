# V0.4 delivery audit

Existing baseline: FastAPI, async SQLAlchemy/Alembic, JWT/RBAC, learner isolation, mock/real vision and LLM adapters, RAGFlow adapter, RAG retrieval/replay, trace, resource review and JSON learning reports. V0.4 adds safe runtime diagnostics, explicit real-service gates, offline replay tooling, report download, PowerShell operations, compose profiles, CI, repository guard and frontend handoff.

External dependencies remain deliberately outside the repository: Docker/MySQL/Redis availability, a RAGFlow URL/key/dataset, model API credentials, local weight path and authorised source documents. No source code, model weight, volume, credential or copyrighted full text is committed.
